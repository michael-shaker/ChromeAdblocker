# Chrome Extension Adblocker

Small Chrome extension that blocks common ad and tracking domains.

## Contents
- `background.js` — blocking rules
- `manifest.json` — extension manifest
- `icons/` — icon files

## Install (developer mode)
1. Open Chrome and go to chrome://extensions
2. Enable Developer mode
3. Click "Load unpacked" and choose this folder

## What it blocks
Blocks requests to a short list of ad and tracking domains (see `background.js`).
